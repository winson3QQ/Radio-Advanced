進階無線電 · 深入數學推導

這是五層學習模式的最後一層（選修）。先在課程頁走完「災害現場 → RF 物理 → SDR 驗證 → 現場應用」，行有餘力想量化計算、預測結果，再到這裡深入公式。

標註約定（事實）＝可查證規格／物理常數；（推論）＝由公式推導；（量級參考）＝受功率／天線／地形影響的粗估，不可當精確值。
情境1 · CTCSS 與 FM 調變
FM調變基本方程： s(t) = A·cos( 2π·f_c·t + (Δf/f_m)·sin(2π·f_m·t) ) 混合後的調變信號： m(t) = V_語音(t) + V_CTCSS(t) V_語音(t) = 語音成分（300–3000Hz帶寬） V_CTCSS(t) = A_sub·sin(2π·f_CTCSS·t) CTCSS 38個標準頻率（EIA規格）： 67.0, 71.9, 74.4, 77.0, 79.7, 82.5, 85.4, 88.5, 91.5, 94.8, 97.4, 100.0, 103.5, 107.2, 110.9, 114.8, 118.8, 123.0, 127.3, 131.8, 136.5, 141.3, 146.2, 151.4, 156.7, 162.2, 167.9, 173.8, 179.9, 186.2, 192.8, 203.5, 210.7, 218.1, 225.7, 233.6, 241.8, 250.3 Hz FM調變指數： β = Δf / f_m （Δf＝頻率偏移，f_m＝調變訊號頻率） 窄帶語音(Δf≈±2.5kHz, f_m≈3kHz)：β ≈ 0.8 →窄帶FM(NBFM) CTCSS亞音：刻意用「很小的偏移」(約±0.5kHz以下)疊加 重點（更正常見誤解）： CTCSS 偏移刻意壓到系統峰值偏移的約15%以下， 只在語音下方佔一小塊頻率，不影響語音可懂度。 CTCSS 能被可靠解碼，靠的是接收端的「窄帶濾波＋連續音判別」， 與 β 大小無關——β 越大只代表佔用頻寬越寬，並不等於越穩定。
情境2 · 接收靈敏度與增益
接收靈敏度公式（事實）： S(dBm) = 10·lg(k·T·B) + NF + SNR_min 各參數說明： k = 波茲曼常數 = 1.38×10⁻²³ J/K T = 標準溫度 = 290 K（約17°C） B = 接收頻寬（Hz） NF = 系統噪聲指數（Noise Figure, dB） SNR_min = 解調所需最低信噪比（dB） 熱噪聲底限計算： 10·lg(k·T) = -174 dBm/Hz （在290K時） 對12.5kHz窄頻FM： 10·lg(k·T·B) = -174 + 10·lg(12500) = -174 + 41 = -133 dBm 加上典型民用接收機參數： NF = 8 dB，SNR_min = 12 dB S = -133 + 8 + 12 = -113 dBm 結論：理想接收靈敏度約 -113 dBm （實際RTL-SDR約 -90 至 -100 dBm） RF增益的dB計算： 增益每提升 6dB = 信號在FFT中高約兩倍 過載點（Overload）：接收機前端飽和 → RTL-SDR典型過載點：約 -30 dBm輸入
情境3 · 頻寬與頻譜效率
頻譜功率密度（推論，基於FM頻譜特性）： 功率密度 PSD(W/Hz) = P_total / B 相同1W發射： 25kHz模式：PSD = 1W / 25000Hz = 40μW/Hz 12.5kHz模式：PSD = 1W / 12500Hz = 80μW/Hz 結論：窄頻的功率密度是寬頻的 2倍（＋3dB） 接收靈敏度公式（重複應用）： S(dBm) = -174 + 10·lg(B) + NF + SNR_min 25kHz時： -174 + 44 + 8 + 12 = -110 dBm （理論） 12.5kHz時：-174 + 41 + 8 + 12 = -113 dBm （理論） 差距：3dB（接收門限低 3dB） 換算成距離要看地形： 自由空間(理想，路徑指數 n=2)：≈ 1.4 倍 實際地形(n≈3~4)：多半只有 1.1~1.2 倍 ※這是「接收雜訊頻寬變小、門限降低」的效果； 並非單純「能量密度」——寬頻 FM 反而有較高的調變增益，是個取捨。 Carson規則（FM「實際佔用頻寬」，事實）： B_FM = 2(Δf + f_max)，f_max＝最高語音頻率(約3kHz) 窄頻：Δf = 2.5kHz → B ≈ 2(2.5+3) = 11kHz（放進 12.5kHz 頻道） 寬頻：Δf = 5kHz → B ≈ 2(5+3) = 16kHz（放進 25kHz 頻道） 重點：12.5 / 25kHz 是「頻道間隔」，不等於實際佔用頻寬； 間隔要比佔用頻寬大一些，留保護帶給鄰頻。
情境5 · 捕獲效應與路徑衰減
FM捕捉效應（Capture Effect）： 當S1/S2 > 6dB（約4倍功率差）時（6dB 為保守教學門檻， 品質好的接收機「捕捉比」可低到 1–3dB） 接收機傾向「捕捉」強信號，弱信號被抑制 → 若干擾信號比我方信號強 > 6dB： 我方信號被完全壓制 ← 無法通話 自由空間路徑衰減（Friis公式）： PL(dB) = 20·log(d) + 20·log(f) + 20·log(4π/c) 簡化版（實際應用，事實推論）： 每倍距離增加約6dB衰減 信號強度推估距離： 假設對講機發射功率5W（37dBm）， 接收靈敏度-100dBm，天線增益0dBi： 最大路徑損耗 = 37 + 0 - (-100) = 137dB 自由空間距離估算（145MHz，僅供量級參考）： ※ 實際距離受發射功率、天線、地形影響甚大，下表只是粗略量級 RSSI S9（強）→ 約 300m–1km RSSI S7（中）→ 約 1–3km RSSI S5（弱）→ 約 3–8km RSSI S3（很弱）→ 約 8–20km 互調干擾產物： 兩個頻率 f1, f2 → 產生 2f1-f2 及 2f2-f1 例：145.500 + 146.000MHz → 產生 145.000MHz虛假信號
情境6 · 菲涅耳區與雙工隔離
菲涅耳區（Fresnel Zone）計算（事實）： 第一菲涅耳區半徑： r1 = √(λ·d1·d2 / (d1+d2)) d1 = 發射端到障礙物距離 d2 = 障礙物到接收端距離 λ = 波長 例：VHF 145MHz，兩點間距3km，中間有山： d1=d2=1.5km，λ=2.07m r1 = √(2.07 × 1500 × 1500 / 3000) ≈ √1552 ≈ 39m → 菲涅耳區需要39m凈空，山丘若超過此高度即造成嚴重衰減 雙工頻率隔離度需求： 中繼台同時收發，發射端信號會干擾接收端 所需隔離度：發射功率 - 接收靈敏度 + 餘量 例：5W(37dBm)發射，接收靈敏度-113dBm： 最低隔離需求 = 37 - (-113) + 20 = 170dB 實際做法（用濾波器）： 每個陷波濾波器（Notch Filter）提供約60–70dB 兩個組成「雙工器（Duplexer）」→ 120–140dB 頻率偏移（Offset）選擇（事實）： VHF民防常見偏移：±600kHz 例：接收145.500，發射146.100MHz UHF常見偏移：±5MHz 例：接收430.000，發射435.000MHz
